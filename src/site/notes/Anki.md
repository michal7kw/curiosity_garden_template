# Anki

## Overview

Anki is a free, open-source flashcard program that uses [[Spaced Repetition]] to optimize memory retention. Unlike simple flashcard apps, Anki schedules reviews based on how well you know each cardâ€”showing difficult cards more often and easy cards less frequently. It's the gold standard tool for anyone serious about long-term memorization.

## Key Information

- **Type:** Software Tool
- **Author:** Damien Elmes
- **First Released:** 2006
- **Platforms:** Windows, Mac, Linux, iOS, Android, Web
- **Cost:** Free (iOS app paid, but AnkiDroid free)

## Why This Resource?

Anki implements [[Spaced Repetition]] algorithms that have been refined over decades of cognitive science research. It's used by medical students, language learners, and anyone who needs to memorize large amounts of information efficiently.

## Main Topics

1. **Spaced Repetition Algorithm**: SM-2 based scheduling that optimizes review timing
2. **Card Types**: Basic, cloze deletion, image occlusion, and custom types
3. **Deck Organization**: Hierarchical organization of flashcard decks
4. **Sync**: Cloud sync across all devices
5. **Add-ons**: Extensive plugin ecosystem for customization

## Summary

### Core Message

Make reviewing a daily habit and let the algorithm handle scheduling. Consistency beats intensity.

### Key Arguments/Points

1. **Daily reviews**: 10-30 minutes daily is more effective than occasional cramming
2. **Make good cards**: One fact per card, test understanding not recognition
3. **Active recall**: The act of retrieving information strengthens memory
4. **Low-overhead learning**: Once set up, the system maintains itself

### Notable Features

- **AnkiConnect**: API for integration with other tools
- **Cloze deletions**: Fill-in-the-blank cards for learning in context
- **Image occlusion**: Hide parts of images (anatomy diagrams, etc.)
- **Custom scheduling**: Advanced users can modify the algorithm
