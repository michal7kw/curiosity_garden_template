# Technical Writing

## Overview

Technical writing is the art of communicating complex information clearly and accurately to a specific audience. Whether writing API documentation, tutorials, or internal guides, this skill is essential for any knowledge worker who needs to share expertise with others.

## Proficiency Levels

### Novice
- Can write basic step-by-step instructions
- Understands the importance of audience awareness
- Uses simple, clear language

### Competent
- Structures documents logically
- Creates effective headings, lists, and formatting
- Anticipates reader questions
- Uses diagrams and visuals when helpful

### Proficient
- Adapts tone and depth for different audiences
- Creates comprehensive documentation systems
- Teaches others to write better documentation
- Edits and improves others' technical writing

### Expert
- Develops documentation standards for organizations
- Creates style guides and templates
- Innovates in documentation formats and tools
- Recognized as a go-to resource for documentation
