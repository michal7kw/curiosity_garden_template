# CS50 Introduction to Computer Science

## Overview

CS50 is Harvard's legendary introduction to computer science, taught by Professor David Malan. Known for its engaging lectures and rigorous problem sets, it covers computational thinking, algorithms, data structures, and web development. The course starts with C to teach low-level concepts, then moves to Python, SQL, HTML/CSS, and JavaScript.

## Learning Objectives

By the end of this course, you will be able to:

1. Think algorithmically and solve problems efficiently
2. Understand abstraction, data structures, and encapsulation
3. Program in multiple languages (C, Python, SQL, JavaScript)
4. Build web applications from scratch
5. Apply [[entities/concept/First Principles Thinking\|First Principles Thinking]] to programming challenges

## Course Structure

### Week 0: Scratch
- Visual programming with MIT Scratch
- Introduction to computational thinking

### Week 1-5: C
- Variables, loops, conditionals
- Arrays, functions, debugging
- Memory, pointers, data structures

### Week 6: Python
- [[entities/skill/Python Programming\|Python Programming]] fundamentals
- Transitioning from C to Python

### Week 7-8: SQL & Web
- Database design
- HTML, CSS, Flask

### Week 9-10: Final Project
- Building a complete application

## Prerequisites

- No prior programming experience required
- Strong problem-solving mindset helpful
- Time commitment is significant (10-20 hours/week)

## Review

**What I liked:**
David Malan's teaching is unparalleled. The production quality is outstanding, and the community (Discord, subreddit) is incredibly supportive.

**What could be improved:**
The jump from C to Python can feel sudden. More gradual transitions between languages would help.

**Would I recommend it?**
100%. This is the gold standard for intro CS courses. Challenging but transformative.
